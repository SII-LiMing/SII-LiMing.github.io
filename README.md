# `lm.github.io` Template

This folder contains a deployable static personal homepage template.

## Local structure

- `index.html`
- `styles.css`
- `main.js`

## How to deploy

The intended deployment target is:

```text
https://lm.github.io
```

The simplest GitHub Pages setup is:

1. Create a GitHub repository named `lm.github.io`
2. Put these files at the repository root
3. Push to the `main` branch

GitHub Pages will then serve the site from the repository root automatically.

## What still needs your real information

To replace the placeholders, I will need:

- your preferred English name on the homepage
- one short biography
- one-line title and affiliation
- email
- links such as Google Scholar / GitHub / CV / LinkedIn
- education history
- selected publications
- projects you want to show
- honors / awards
- optionally a portrait photo
